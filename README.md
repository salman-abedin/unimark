# unimark: Browser Independent Bookmark Utility

# Dependencies

-  amenu/dmenu/rofi
-  ydotool
-  wl-clipboard/xsel/xclip
-  grep, awk, sed, tr, sort, uniq, sleep & xargs

# Supported Browsers

- Qutebrowser
- Brave
- Firefox

## Installation

```sh
git clone https://github.com/salman-abedin/unimark.git && cd unimark && sudo make install
```

## Uninstallation

```sh
sudo make uninstall
```
